# Motion Reference Index

Sourced 2026-05-29. Contains 66 indexed online references. Use this as a lookup table before generating AI in-between frames. Prefer public-domain and open references, and check the linked item license before copying or redistributing any media. For most animation work, extract pose, timing, contact, and silhouette facts rather than copying the source image.

Columns: `domain`, `subject_type`, `action_or_pose`, `pose_tags`, `source_family`, `source_title`, `source_url`, `media_type`, `rights_or_license`, `use_for`, `notes`.

## Quick Filters

- Horse/quadruped gait: filter `subject_type` for `horse`, `dog`, `cat`, `goat`, `camel`, `elephant`, `lion`, `deer`, `pig`, `buffalo`, or `general quadruped`.
- Human body mechanics: filter `domain=human` and action terms like `walk`, `run`, `jump`, `throw`, `dance`, `carry`, `turn`, `hammer`, or `object`.
- Rotoscope-style frame sequences: prefer `source_family` containing `Muybridge`, `Marey`, `Library of Congress`, `Wikimedia Commons`, `NGA`, `Smithsonian`, or `Biodiversity Heritage Library`.
- 3D pose/timing: prefer `source_family` containing `CMU`, `HDM05`, `KIT`, `PFERD`, or `SyDog`.
- Timing/spacing revisions: filter `domain=principle` or `domain=effects`.
